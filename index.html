What is machine learning?

Machine learning refers to a type of data analysis that uses algorithms that learn from data. It is a type of artificial intelligence (AI) that provides systems with the ability to learn without being explicitly programmed. This enables computers to find data within data without human intervention.

How can I get started?

With CSynapse.com of course! Check out the tutorial below for a more detailed overview on how.

Create a CSynapse

A CSynapse is named dataset that can have various machine learning algorithms attatched to it. All you need to do to create a CSynapse is give it a name and your tagged data you want analyzed.

Add algorithms to your CSynapse.

There are a ton of machine learning algorithms that all work in different ways. Using the drop down menu, add multiple kinds to figure out which ones will work best with your data.

Analyze your results.

CSynapse automatically displays information about your data in a way that makes comparison a breeze. Cross validation is used to analyze the accuracy of each algorithm and regression data is provided to help you understand how the results were found. Charts and plots are also provided to give even easier insight into your data.

Refine your favorite algorithm.

Now that you know which algorithm is best, you can click the "Add more algorithms..." link to add an aditional copy of that algorithm to your CSynapse. When doing this, you can also change the parameters associated with that algorithm to potentially improve your analysis even further.

Classify your untagged data.

Now that you have found the best algorithm for your data, simply click on that algorithm from the results page to create a new classification. Give this new classification a name, and upload data with no tags at all to begin making predictions on it.

Review your classifications.

When your classification completes, you will be able to access it through the classifciations panel on the dashboard, the dropdown on the top bar, or the classifcations page. You classification will be a copy of your provided data, with classification labels appended. It's that easy!

What are you waiting for?

Now you have all the information you need to harness the power of machine learning. Get started today!
